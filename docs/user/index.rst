User Guide
==========

DBall Application is designed to become user friendly, simple and clean. Any type of user no matter what level of their
computer skills is targeted for this application. Addition to its simple design, it is designed to be multi-functional.
More of its functions such as altering and registering new record available through registration. Still all of the record
can be accessible through front view. DBall also provides a abstract interface for developers. With its robust REST API,
developers can use our services in their programs easily. For further information about this topic please advance
to :ref:`dev-guide`.
